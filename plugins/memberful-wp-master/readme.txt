== Description ==

Integrate Memberful with WordPress using the Memberful OAuth 2.0 endpoint.
 
== Installation ==
 
1. Visit Plugins => Add New => Upload in your WordPress dashboard.
2. Upload and activate the plugin.
3. Activate the WordPress service from your Memberful account.
4. Paste in your registration key and click the "Connect to Memberful" button.
